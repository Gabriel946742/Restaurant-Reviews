# Restaurant-Reviews
--PROYECTO DE RESEÑAS (ES)---

Este proyecto es una aplicación web diseñada para la recolección de reseñas de un restaurante ubicado en Torrevieja. La plataforma permite a los usuarios dejar valoraciones mediante un sistema de estrellas, mientras que las reseñas son enviadas directamente al jefe del restaurante para su revisión. La web está desarrollada utilizando HTML, CSS, PHP y JavaScript, y cuenta con un sistema multilingüe, disponible en español, inglés, francés y ruso. El objetivo es ofrecer una manera sencilla y eficaz de recibir comentarios de los clientes para mejorar la calidad del servicio

---MANUAL BÁSICO---

Puedes revisar la web activa en resenasqueenmisisipi.com o descargar toda la web y ejecutarla en LOCALHOST mediante XAMPP.


---PROYECTO DE RESEÑAS (ING)---

This project is a web application designed for collecting reviews for a restaurant located in Torrevieja. The platform allows users to leave ratings through a star system, with the reviews being sent directly to the restaurant manager for review. The website is built using HTML, CSS, PHP, and JavaScript, and features a multilingual system available in Spanish, English, French, and Russian. The goal is to provide a simple and effective way to receive customer feedback to improve the quality of service.

---BASIC MANUAL---

You can review the live website at resenasqueenmisisipi.com or download the entire website and run it locally on LOCALHOST using XAMPP.
